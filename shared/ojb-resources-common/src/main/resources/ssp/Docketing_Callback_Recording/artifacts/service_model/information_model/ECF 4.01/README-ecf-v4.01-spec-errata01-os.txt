This package contains the document ecf-v4.01-spec-errata01-os
approved by the OASIS LegalXML Electronic Court Filing TC
on 14 July 2014, and the companion document
ecf-v4.01-spec-errata01-os-complete, a version of the
Electronic Court Filing Version 4.01
OASIS Standard incorporating the Approved Errata. These
two files are provided together to assist you in performing a
complete review of the Approved Errata.
